# DAppKit Next.js App

A Next.js 14 application demonstrating the [DAppKit](../../packages/dappkit) library for building decentralized applications.

## Features

- **Next.js 14** with App Router
- **Wallet Connection** via RainbowKit (MetaMask, WalletConnect, Ledger, Safe)
- **Multi-chain Support**: Ethereum, BSC, Polygon, IoTeX
- **Type-safe APIs** with tRPC
- **Authentication** with NextAuth.js
- **UI Components** with NextUI and Tailwind CSS
- **State Management** with MobX

## Getting Started

```bash
# Install dependencies
pnpm install

# Configure environment variables
cp .env.example .env
# Edit .env with your configuration

# Run development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
